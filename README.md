# 🌱 Agro Forte: Futuro Sustentável

> Concurso Agrinho 2026 – Programação Front-End (HTML, CSS e JavaScript)

## 📌 Sobre o Projeto

**Agro Forte: Futuro Sustentável** é um site educativo e interativo desenvolvido para o Concurso Agrinho 2026 com o tema:

> **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente."**

O projeto busca conscientizar os visitantes sobre a importância da produção agrícola responsável, demonstrando que é possível aumentar a produtividade no campo utilizando tecnologias e práticas sustentáveis que preservem os recursos naturais para as futuras gerações.

O site apresenta conteúdos educativos, recursos interativos e ferramentas que incentivam a reflexão sobre escolhas sustentáveis no meio rural.

---

## 🎯 Objetivos do Projeto

- Incentivar a valorização do agronegócio sustentável;
- Demonstrar como a tecnologia pode contribuir para a preservação ambiental;
- Promover a educação ambiental de forma acessível e interativa;
- Estimular a participação dos usuários por meio de atividades práticas;
- Desenvolver habilidades em HTML, CSS e JavaScript conforme o regulamento do Concurso Agrinho.

## ▶️ Como Executar o Projeto

### Opção 1 – Localmente

1. Faça o download do projeto;
2. Extraia os arquivos;
3. Abra a pasta do projeto;
4. Clique duas vezes no arquivo:

```
index.html
```

O site será aberto no navegador padrão.

---

### Opção 2 – GitHub Pages

1. Envie todos os arquivos para um repositório público no GitHub;
2. Vá em:

```
Settings → Pages
```

3. Em "Build and deployment", selecione:

```
Deploy from a branch
```

4. Escolha:

```
Branch: main
Folder: /root
```

5. Salve as alterações.

O GitHub gerará automaticamente o link público do projeto.
